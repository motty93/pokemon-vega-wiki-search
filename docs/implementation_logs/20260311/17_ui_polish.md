# UI細かい調整まとめ

## 変更内容

### トップページのグラデーション統一
- ダーク縞々背景を詳細ページヘッダーと同じ indigo/purple 系グラデーションに変更
- 縞パターンは `rgba(255,255,255,0.06)` の半透明白で残し、ベースカラーを統一

### 検索動作の変更
- keyup即時発火を削除し、Enter or 検索アイコンクリック時のみ発火に変更
- 詳細フィルター（タイプ選択、種族値フィルター）を全削除、あいまい検索のみに

### スティッキーヘッダー
- 検索結果がhtmxで挿入されたタイミングでヘッダーがスライドイン
- `position: fixed; top: -60px` → `.visible` クラスで `top: 0` に遷移
- 結果テンプレート側に `pt-20` を付与してヘッダーとの被りを回避
- 結果が空のときはpaddingなし（ファーストビューに空白が出ない）

### 画像サイズ調整
- 一覧: `w-20 h-20` → `w-40 h-40`（160px）、背景枠なしで直接表示
- 詳細: `w-48 h-48` → `w-60 h-60`（240px）

### その他
- 一覧からTotal（合計種族値）表示を削除
- `.type-badge` の `@apply` をやめて直接CSSに（Tailwind CDNでの `@apply` 不具合回避）
- テキスト色をグラデーション背景に合わせて `text-white/*` 系に調整

### ロギング + Air
- `internal/handler/middleware.go`: リクエストログミドルウェア追加（メソッド、パス、ステータス、レスポンス時間）
- `cmd/server/main.go`: `LoggingMiddleware(mux)` でミドルウェア適用
- `.air.toml`: Air設定追加（`.go` `.html` 監視、自動リビルド）
- `Makefile`: `make dev` を `air` に変更
